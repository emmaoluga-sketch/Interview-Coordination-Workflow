# Interview Coordination Workflow

> This repository demonstrates a scalable interview coordination system designed to reduce delays and improve candidate experience.

## Problem
- Scheduling conflicts
- Long delays between interview stages
- Poor candidate experience

## Solution
- Centralized scheduling system
- Calendar pooling
- Structured interview kits
- Candidate tracking system

## Tools
- Greenhouse
- GoodTime
- Calendly
- Slack

## Outcomes
- 65% reduction in scheduling time
- 4.8/5 candidate satisfaction
- +18% offer acceptance rate
